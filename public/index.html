Acerca del Proyecto

Este sistema integral de gestión de refaccionaria (`Refaccionaria Gallegos`) fue desarrollado por estudiantes de la Universidad Autónoma de Aguascalientes (UAA), cursando el Séptimo Semestre de Ingeniería en Informática.

El proyecto surge como una iniciativa para aplicar conocimientos avanzados en Bases de Datos NoSQL (MongoDB), desarrollo Backend (Node.js/Express) y diseño Frontend interactivo.

Objetivos

  • Arquitectura Full Stack robusta.
  • Experiencia de usuario óptima.
  • Funcionalidades avanzadas (CRUD, Rastreo).

Política de Privacidad

En `Refaccionaria Gallegos`, la privacidad es fundamental.

1. Datos Recopilados: Recopilamos nombres y correos solo para simulación académica.

2. Uso de Información: Datos usados exclusivamente en BD local para simulación.
